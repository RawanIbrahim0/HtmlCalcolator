# HtmlCalcolator
أول شي اخترت التصميم - اخدتو نفسو تبع جوالي
عملت شبكة من الأزرار وصفيتهن بالترتيب
بعدين نسقت الشبكة بملف الستايل حطيتلها خلفية وعملتلها شوية radius
ونسقت بعدين شكل الأزرار حسب الديزاين نفسو اللي عندي
الأزرار حطيتهن button والشاشة حطيتها div عادية
كل الأزرار عطيتهن نفس الكلاس لياخو نفس التنسيق الا الزرار المميزة اللي هي تبع العمليات حطيتهن بكلاس لحالهن مشان اعطيهن نفس التنسيق للباقي بس عليهن شي مميز متل عندي لون الخط كان برتقالي
للdiv تبع الشاشة عطيتو الخلفية transparent مشان تكون شفافة
